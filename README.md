# Cisco
# Projeto Redes - Belém e Castanhal

Projeto acadêmico de interligação de duas cidades utilizando:

- VLANs
- Router-on-a-Stick
- Roteamento Estático
- ACLs
- SSH
- Port Security

## Endereçamento

### WAN
200.200.200.0/24

### Belém
VLAN10: 192.168.10.0/24
VLAN20: 192.168.20.0/24

### Castanhal
VLAN10: 192.168.110.0/24
VLAN20: 192.168.120.0/24

## Segurança aplicada

- SSH
- Senhas criptografadas
- Port Security
- ACLs restritivas

### - SEQUENCIA

- Switch

- ACL
